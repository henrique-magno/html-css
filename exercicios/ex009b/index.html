Para que utilizamos listas?

Listas ordenadas com a função OL, ordened list + LI de list

A ordem faz diferença
  1. acordar
  2. Beber água
  3. Comer
  4. Dormir novamente
Assim ficará mais fácil de adicionar alterações na lista e manter a numerção ordenada
Agora para alterar de numeração, utilize o comando TYPE (A, I, a, i) / START (números posteriores) ao lado de OL, veja:
  1. acordar
  2. Beber água
  3. Comer
  4. Dormir novamente
